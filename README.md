Разработана на **Node.js** с использованием фреймворка **Express**. Обеспечивает работу клиентского приложения и панели администратора.

##  Технологический стек
*   **Runtime:** Node.js
*   **Framework:** Express.js
*   **Database:**  MySQL
*   **Auth:** JWT
*   **Архитектура:** MVC-pattern (Routes -> Controllers -> Models)

##  Структура проекта

*   `controller/` — Бизнес-логика приложения:
    *   `userController.js` — Регистрация, вход, управление пользователями (блокировка).
    *   `ticketController.js` — Покупка и валидация билетов.
    *   `attractionController.js` — CRUD для аттракционов.
    *   `ReportController.js` — Генерация статистики и отчетов.
*   `models/` — Схемы базы данных (описание таблиц).
*   `routes/` — Маршрутизация API запросов.
*   `middleware/` — Промежуточное ПО (проверка авторизации, обработка ошибок).
*   `db.js` — Конфигурация подключения к базе данных.

##  Ключевой функционал API

1.  **Авторизация и Безопасность:**
    *   Регистрация и логин с выдачей JWT.
    *   Разграничение прав доступа (Role-based access): Пользователь / Администратор.
2.  **Биллинг:**
    *   Пополнение баланса пользователя.
    *    покупка билетов.
3.  **Управление парком:**
    *   Добавление/удаление аттракционов.
    *   Управление ценами.
4.  **Отчетность:**
    *   Расчет посещаемости за период.
    *   Финансовая статистика.
